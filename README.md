# My First Rhythm Game

Minimaler Rhythm-Game-Prototyp in Godot (4.x).

## Start
1. Godot 4.x öffnen.
2. Projektordner auswählen (`my-first-rhythm-game`).
3. Szene `scenes/Main.tscn` starten (Play).

## Steuerung
- **Space/Enter (ui_accept)**: Note treffen, wenn sie die Hit-Line erreicht.

## Gameplay-Notizen
- Notes spawnen automatisch und fallen nach unten.
- Trefferfenster und Geschwindigkeit lassen sich im Script `scripts/Main.gd` anpassen.
